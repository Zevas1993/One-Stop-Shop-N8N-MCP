Diagnostic Instructions

How to diagnose the rendering issue:
  1. Enter your n8n API key above
  2. Click "Run Diagnostic Tests" to fetch and analyze the workflow
  3. Click "Open Workflow in n8n UI" to open it in a new tab
  4. While that tab is open, press F12 to open browser DevTools
  5. Go to the Console tab
  6. Look for any RED error messages
  7. Copy those errors and share them
